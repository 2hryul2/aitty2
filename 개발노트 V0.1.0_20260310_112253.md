## 상세 변경 내용

### 1. 헤더/브랜딩 화면 조정 (`App.tsx`, `App.css`)

- 제목을 한 줄 구조로 바꾸고 `신한DS` 강조 스타일을 추가함
- 상단 여백과 글자 크기를 줄여 실제 터미널 표시 영역을 넓힘
- 헤더 테두리/그림자 스타일을 단순화해 가독성을 높임

### 2. AI 터미널 상태/조작 개선 (`AITerminal.tsx`)

- 설정 변경 추적 상태 추가
- `isDirty`: 설정이 바뀌었지만 아직 적용되지 않은 상태
- `isApplySuccess`: 최근 적용 성공 여부
- `Apply` 실행 결과(성공/실패)를 버튼 시각 상태와 메시지로 명확히 표시
- 시스템 프롬프트 입력 영역을 토글 방식으로 전환
- 상단 도구 영역에 `AI분석` 버튼 배치
- 초기 진입 시 제공자/엔드포인트/모델/연결 여부를 바로 출력
- 30초 주기 AI 상태 점검(헬스체크) 추가
- 응답 대기 중 랜덤 안내 문구 + 타이프라이터 효과 추가
- 드래그 선택 자동 복사, 우클릭 붙여넣기 지원

### 3. SSH 터미널 안정성/편의성 강화 (`SSHTerminal.tsx`)

- IPC 기반 30초 주기 SSH 연결 상태 점검 추가
- 연결 끊김 감지 시 폴링/상태 점검 중단, 경고 출력, 배너 복구, 연결 폼 재노출
- 드래그 선택 자동 복사, 우클릭 붙여넣기 기능 추가
- 언마운트/수동 연결 해제 시 타이머 및 이벤트 정리 강화

## 영향 및 점검 항목

- 기대 효과
- 상태 가시성이 좋아져 문제 인지 속도 향상
- 복사/붙여넣기 개선으로 반복 작업 효율 증가

- 확인 필요
- WebView2 환경에서 클립보드 권한 동작 여부
- AI 응답 취소 시 진행 표시 종료 타이밍
- 30초 주기 상태 점검이 성능/로그에 미치는 영향

## 점검 체크리스트

- `Apply` 버튼 상태 변화(미적용/성공) 정상 동작 확인
- AI `Check`/`Apply` 후 상태 메시지 정확성 확인
- `AI분석` 실행 시 진행 안내 및 결과 출력 확인
- SSH 강제 끊김 시 경고/배너 복구/연결 폼 전환 확인
- 자동 복사 및 우클릭 붙여넣기 동작 확인
