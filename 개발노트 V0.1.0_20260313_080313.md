## 3. `AITerminal.tsx` — 핵심 변경 5건

### 3-1. isDirty / isApplySuccess 상태 추가 (L38-39)

설정 변경 후 적용 여부를 버튼 색상으로 즉시 시각화.

```tsx
const [isDirty, setIsDirty] = useState(false)        // 미적용 설정 변경 감지
const [isApplySuccess, setIsApplySuccess] = useState(false)  // Apply 성공 여부
```

**Apply 버튼 상태 매핑**

| 조건 | 버튼 색상 |
|------|---------|
| 설정 변경 후 미적용 (`isDirty=true`) | 황색 (`#ffc107`) |
| Apply 성공 (`isApplySuccess=true`) | 녹색 (`#4caf50`) |
| 기본 | 기본 스타일 |

트리거: Endpoint/API Key/SystemPrompt 변경 시 `isDirty=true`, `isApplySuccess=false`

### 3-2. 초기 설정 점검 (L479-506)

WebView2 환경에서 앱 시작 시 설정 상태를 터미널에 자동 출력.

```
■ 초기 설정 점검 중...
├─ Provider : Ollama (Local)
├─ Endpoint : http://172.16.1.103:11434
├─ Model    : qwen2.5-coder:7b
└─ ✓ 연결 성공 | 바로 AI에게 질문하세요!
```

연결 성공 시 `isApplySuccess=true` 설정 → Apply 버튼 초기 녹색 표시.

### 3-3. 타이프라이터 대기 메시지 (L229-258, L389-419)

`sendMessage` + `handleAnalyzeClick` 양쪽 동일 로직 적용.

3. 애니메이션 완료 시 버퍼링된 청크 일괄 flush

```tsx
// 드래그 선택 → 클립보드 자동 복사

})

// 우클릭 → 클립보드에서 붙여넣기 → inputBufferRef에도 추가
const handleContextMenu = (e: MouseEvent) => {
  e.preventDefault()
  if (isProcessingRef.current) return
  navigator.clipboard.readText()
    .then(text => {
      if (text) { term.write(text); inputBufferRef.current += text }
    })
    .catch(() => {
      term.writeln('\r\n\x1b[33m⚠ 클립보드 권한 없음. Ctrl+V를 사용하세요.\x1b[0m')
      writePrompt()
    })
}
```

## 4. `SSHTerminal.tsx` — 핵심 변경 3건

### 4-1. 30초 SSH 헬스체크 (L82-108)

```tsx
const startHealthCheck = useCallback((showBannerFn) => {
  healthCheckTimerRef.current = setInterval(async () => {
    const { isConnected } = await sshBridge.state()
    if (!isConnected) {
      stopHealthCheck(); stopPolling()
      term.writeln('\r\n\x1b[31m⚠ SSH 연결이 끊어졌습니다.\x1b[0m')
      setTimeout(() => { term.clear(); showBannerFn(term) }, 2000)
      setShowConnectForm(true)
      onDisconnect?.()
    }
  }, 30_000)
}, [stopHealthCheck, stopPolling, onDisconnect])
```

**타이머 수명 관리**

| 이벤트 | 동작 |
|--------|------|
| 연결 성공 | `startHealthCheck(showBanner)` |
| 수동 해제 | `stopHealthCheck()` + `stopPolling()` |
| 끊김 감지 | `stopHealthCheck()` + `stopPolling()` + 배너 복구 |
| 언마운트 | `useEffect` cleanup → `stopPolling()` + `stopHealthCheck()` |
