## 5) 핵심 Q/A

### Q1. `aitty2.git` 연결된 작업 진행 상태는?
A1. `D:\source\aitty2`가 `https://github.com/2hryul2/aitty2.git`에 연결되어 있고, `master`가 `origin/master`와 동기화된 상태였습니다. 미커밋 변경/미푸시 커밋은 없었습니다.

### Q2. 최근 커밋 기준으로 무엇이 바뀌었나?
A2. 가장 큰 변화는 클라우드 Claude 기반 AI 경로를 로컬 Ollama 기반으로 교체한 점입니다. 여기에 UI 리사이즈/배너 개선, 초기 MVP 아키텍처 구축이 이어집니다.

### Q3. 전체 코드는 몇 라인인가?
