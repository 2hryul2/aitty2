## 🔧 주요 구현 로직

### SSH 연결 흐름
```
사용자 입력 (Host/Port/User/Password)
    ↓
SSHTerminal.tsx: handleConnect()
    ↓
IpcHandler.Connect (C#)
    ↓
SshService.ConnectAsync()
    ↓
SSH.NET: AuthenticateAsync() [비밀번호 or 개인키]
    ↓
OpenShellStream() → PTY 할당
    ↓
startPolling() → 100ms마다 shellRead()
    ↓
터미널에 출력 렌더링 (xterm.js)
```

### AI 분석 흐름
```
SSH 명령 실행 & 출력 표시
    ↓
[AI분석] 버튼 클릭
    ↓
SSH 마지막 출력 캡처 (SshService.GetLastOutput)
    ↓
AITerminal에 프롬프트 자동 생성
    ```
    "docker logs app 명령어 결과를 분석해줘:
    [에러 로그...]"
    ```
    ↓
IpcHandler.SendAiRequest()
    ↓
선택된 Provider에 따라 라우팅:
├─ Ollama → LocalLlmService.SendRequest()
└─ Gemini → GeminiService.SendRequest()
    ↓
스트리밍 응답 → AITerminal 렌더링
```

### 멀티 AI 프로바이더 라우팅
```csharp
// IpcHandler.cs
private string _currentProvider = "ollama"; // 기본값

case "ai.set-provider":
    _currentProvider = (string)args["provider"];
    break;

case "ai.send-request":
    if (_currentProvider == "gemini")
        await _geminiService.SendRequest(...);
    else if (_currentProvider == "ollama")
        await _localLlmService.SendRequest(...);
    break;
```
