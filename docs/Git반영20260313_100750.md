## 주요 변경 내용

### 1. Claude 제공자 지원 추가

- `AiServiceManager`에 `claude` 제공자 등록
- API 키 저장/상태 조회/제공자 목록 반환에 `claude` 반영
- 서비스 종료 시 `ClaudeApiService`도 함께 정리되도록 처리

### 2. Claude 서비스 공통화

- `ClaudeApiService`가 `IAiService`를 구현하도록 변경
- 현재 제공자명, 설정 여부, 모델 목록 조회 등을 공통 방식으로 제공
- 기본 모델 및 알려진 모델 목록을 내부에서 관리하도록 정리

### 3. AI 터미널 동작 개선

- `gemini` 전용 분기를 API Key 기반 제공자 공통 로직으로 확장
- 제공자별 표시 이름과 상태 메시지를 동적으로 출력
- 분석/대화 스트리밍 시:
- 서버 요청은 먼저 시작
- 초기 안내 배너는 병렬 출력
- 배너 출력 중 도착한 응답은 버퍼링 후 이어서 출력
- 앱 시작 시 제공자/엔드포인트/모델/연결 상태 점검 메시지 보강
- 클립보드 권한이 없을 때 안내 문구를 출력하도록 개선

### 4. SSH 터미널 안내 보강
